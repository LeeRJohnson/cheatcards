rel microformats

rel-license
Add rel="license" to a hyperlink when the destination contains the content license for the linking document.
rel-tag
Add rel="tag" to a hyperlink when the destination describes the content in the linking document and the final part of the destination URL is a descriptive keyword.

XFN

Space-separated values of the rel attribute on a hyperlink.

  • friend, acquaintance, contact
  • met
  • co-worker, colleague
  • co-resident, neighbour
  • child, parent, sibling, spouse, kin
  • muse, crush, date, sweetheart
  • me
<a href="http://adactio.com/" rel="friend met colleague">Jeremy Keith</a>

hCard

vcard
  • fn
  • n (can be inferred from fn)
    • family-name
    • given-name
    • additional-name
    • honorific-prefix
    • honorific-suffix
  • url, email, tel, adr
  • title, role, org
  • nickname
  • photo (src attribute of img)
  • bday (ISO 8601)
<cite class"vcard"><a href="http://adactio.com/" class="fn url">
Jeremy Keith
</a></cite>

adr

adr
  • post-office-box
  • extended-address
  • street-address
  • locality
  • region
  • postal-code
  • country-name
<ol class="adr">
<li class="street-address">
28 Kensington Street</li>
<li class="locality">
Brighton</li>
<li class="postal-code">
BN1 4AJ</li>
</ol>

hCalendar

vcalendar
  • vevent
    • summary
    • dtstart (ISO 8601)
    • dtend (ISO 8601)
    • location (hCard, adr)
    • url
    • description
<p class"vevent">
There's a
<span class="summary">
Salter Cane concert
</span> in
<span class="location">
The Prince Albert
</span> on
<abbr class="dtstart" title="2010-06-04">
Friday</abbr>.</p>
